# E-Commerce Website

## Overview
This project is a simple E-Commerce website that allows users to browse products, view product details, add items to a cart, and view offers. The website is built using HTML, CSS, and JavaScript.

## Project Structure
```
ecommerce-website
├── index.html
├── products.html
├── product-details.html
├── cart.html
├── contact.html
├── offers.html
├── css
│   └── style.css
├── js
│   ├── script.js
│   ├── cart.js
│   └── products.js
└── README.md
```

## Features
- **Homepage**: Displays a Featured Products section with product cards that link to the product details page.
- **Products Page**: Shows all available products in a grid format with product cards.
- **Product Details Page**: Provides detailed information about a specific product, including an "Add to Cart" button.
- **Cart Page**: Displays products added to the cart, including quantity and total price calculations.
- **Contact Page**: Contains contact information for the business.
- **Offers Page**: Lists current offers or discounts available on the website.

## Setup Instructions
1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to view the homepage.
3. Navigate through the website using the provided links to explore products, view details, and manage the cart.

## Technologies Used
